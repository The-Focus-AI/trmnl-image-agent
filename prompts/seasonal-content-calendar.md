# Cornwall CT Seasonal Content Calendar

Configuration for rotating TRMNL dashboard content throughout the year.

## How to Use This Calendar

1. Check the current date
2. Find the matching date range below
3. Use the specified theme, data sources, and imagery prompts
4. School calendar events override seasonal themes when relevant

## Integration with Dashboard

### Priority Order for Banner Content

1. **Weather emergencies** (storms, extreme cold) - ALWAYS top priority
2. **School closures/delays** (snow days, emergencies)
3. **School calendar events** (holidays, breaks, early dismissals within 3 days)
4. **Seasonal/holiday themes** (from this calendar)
5. **Default seasonal imagery**

### Combining School + Seasonal Content

When a school event coincides with a seasonal theme, combine them:

```
School: NO SCHOOL MON - MLK DAY
Seasonal: Peak ski season

Combined banner: "NO SCHOOL MON â€¢ PERFECT SKI CONDITIONS"
```
